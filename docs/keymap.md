## 겹모음 (연속 입력)

- ㅗ + ㅏ → ㅘ
- ㅗ + ㅐ → ㅙ
- ㅗ + ㅣ → ㅚ
- ㅜ + ㅓ → ㅝ
- ㅜ + ㅔ → ㅞ
- ㅜ + ㅣ → ㅟ
- ㅡ + ㅣ → ㅢ

## 받침/겹받침 (연속 자음)

- ㄱ + ㅅ → ㄳ
- ㄴ + ㅈ → ㄵ
- ㄴ + ㅎ → ㄶ
- ㄹ + ㄱ → ㄺ
- ㄹ + ㅁ → ㄻ
- ㄹ + ㅂ → ㄼ
- ㄹ + ㅅ → ㄽ
- ㄹ + ㅌ → ㄾ
- ㄹ + ㅍ → ㄿ
- ㄹ + ㅎ → ㅀ
- ㅂ + ㅅ → ㅄ

## 메모

- 대문자는 기본 모음과 동일 취급. (O/P만 ㅒ/ㅖ)
- 겹받침 분리 규칙: 일반 분리 표준에 맞춤.(앞자음 유지)
