## 상태 패턴 적용을 위한 DFA

### 상태

- 초성만 있는 상태: A
- 초성 + 중성이 있는 상태: B
- 초성 + 중성이 있는 상태(중성이 합성모음인 경우): B1
- 종성까지 있는 상태: C
- 종성까지 있는 상태(종성이 겹자음인 경우): C1
- 초성, 중성, 종성 모두 비어있는 상태: E

## 입력

- 자음
- 모음
- 기타(특수문자, 숫자 등)
- 용어: "확정"은 해당 값을 누적 결과에 추가하는 것을 의미함
- 조합/분해 판단은 별도 규칙 테이블(모음/받침 조합, 겹받침 분해)에 위임한다

### 상태 전환

- A
  - 자음 입력 => A
  - 모음 입력 => B
  - 기타 입력 => E(모든 값 확정 후 상태 비움)

- B
  - 자음 입력 => C
  - 모음 입력
    - 모음이 합성 불가능한 경우 => B
    - 모음이 합성 가능한 경우 => B1
  - 기타 입력 => E(모든 값 확정 후 상태 비움)
- B1
  - 자음 입력 => C
  - 모음 입력 => B(이전 입력 확정 후 B로 시작, 초성은 'ㅇ' 채움)
  - 기타 입력 => E(모든 값 확정 후 상태 비움)

- C
  - 자음 입력
    - 받침이 합성 불가능한 경우 => A
    - 받침이 합성 가능한 경우 => C1
  - 모음 입력 => B(받침을 다음 초성으로 넘김)
  - 기타 입력 => E(모든 값 확정 후 상태 비움)
- C1
  - 자음 입력 => A(겹받침 확정 후 새 초성 시작)
  - 모음 입력 => B(겹자음 분해 후, 남은 값 확정)
  - 기타 입력 => E(모든 값 확정 후 상태 비움)

- E
  - 자음 입력 => A
  - 모음 입력 => B
  - 기타 => E(모든 값 확정 후 상태 비움)
